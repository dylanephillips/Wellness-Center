# Wellness Center Web Application

## Overview
The Wellness Center Web App allows users to register, log in, book wellness services, and lets administrators manage services and users.  
Built with **PHP (OOP & MVC), MySQL, HTML/CSS**, and runs on a local server like XAMPP.
